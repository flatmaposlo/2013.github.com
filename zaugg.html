What's new in Scala 2.10? Session

Scala 2.10 arrived in January, bringing with it some every day conveniences like the standardized Futures API, String Interpolation, Value Classes, and Implicit Classes, along with the power features of Dynamic, Def Macros and Reflection. Learn how these features work together, and see how they are being used to provide the next generation of expressive, performance, and type safe libraries.

Welcome to (Macro) Paradise Workshop

We'll examine macros in detail: when are they appropriate, how they are invoked by the compiler, and how you can implement them. This journey will take us from the standard Scala 2.10 release (which includes, experimentally, def-macros), through to the fabled "Macro Paradise," where R&D is underway to provide Type Macros and more.

Concretely, we will:

  • Learn how to reflect on types
  • Use the REPL to experiment with compiler internals
  • Build a macro, step-by-step, to transform a block of code
  • Use reify, manual tree creation, and the new quasiquotes to construct and pattern match against trees.
  • Learn what it means to be sym-ful
  • Understand how to write (or use pre-written) tree Traversers and Transformers.

Prerequisites:

  • Basic knowledge of git (or be sitting next to someone with said knowledge)
  • You've at least browsed through http://docs.scala-lang.org/overviews/reflection/overview.html and the "Def Macros" section http://docs.scala-lang.org/overviews/macros/overview.html
  • SBT 0.12.3 installed (http://scala-sbt.org)
