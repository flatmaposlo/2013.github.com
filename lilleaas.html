WTF is Datomic? Session

Datomic is a new database with a twist on client/server and the change of data over time. For example, all data is immutable. And queries run on the client, not the server. Wat!!

  • I'll tell you why (and when) that's a good thing.
  • I'll introduce what Datomic is and how it works. Datomic is pretty different from most of the other databases out there.
  • I'll explain how to use Datomic to store and query your data,
  • I'll showcase some of the uses for datomic other than being the database in traditional server apps.
