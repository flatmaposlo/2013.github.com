Flatmap 2013
============

It's the Flatmap site!

Prerequisites
-------------

Node >= 0.8.x.

If `npm install -g` requires root access, install Grunt manually:

```bash
$ sudo npm install -g grunt
```

Jekyll


```bash
$ gem install jekyll
```



Setup
-----

```bash
$ make
```

Start local server
------------------

```bash
$ make server
```
